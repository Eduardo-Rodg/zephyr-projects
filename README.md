# XIAO nRF54L15 Zephyr Projects

Repositorio que contiene las pruebas, ejemplos y proyectos desarrollados utilizando la placa Seeed Studio XIAO nRF54L15 y el sistema operativo en tiempo real Zephyr RTOS.

## Objetivo

Este repositorio sirve como espacio de aprendizaje y experimentación para familiarizarse con el desarrollo de aplicaciones embebidas en Zephyr, incluyendo el uso de periféricos, Bluetooth Low Energy (BLE), sensores y servicios personalizados.

## Proyectos

### Blinky
Ejemplo básico para controlar el parpadeo de un LED utilizando GPIO.

### button
Lectura del estado de un botón y manejo de eventos de entrada.

### peripheral_lbs
Implementación del servicio LED Button Service (LBS) mediante Bluetooth Low Energy.

### peripheral_external_lbs
Versión modificada del servicio LBS para controlar un LED externo conectado a la tarjeta.

### sensor_data
Adquisición y procesamiento de datos provenientes de sensores.

### sensor_data_CSV
Proyecto para almacenar o exportar datos de sensores en formato CSV.

### sensor_data_ble
Transmisión de datos de sensores mediante Bluetooth Low Energy.

## Herramientas utilizadas

- Seeed Studio XIAO nRF54L15
- Zephyr RTOS
- nRF Connect SDK
- Visual Studio Code
- Nordic Toolchain
- Bluetooth Low Energy (BLE)
